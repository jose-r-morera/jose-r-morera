### 1. FinanceFlow: Agentic AI Financial Outlook Analyzer (_Under Development_)
> A full-stack AI system designed to predict a company's 1-year financial outlook (Positive, Neutral, or Negative) by analyzing SEC filings (10-K, 10-Q, 8-K) through a Retrieval-Augmented Generation (RAG) pipeline. 
> 
> The architecture leverages the Model Context Protocol (MCP) and LangGraph to orchestrate complex reasoning workflows. It utilizes a router-worker model with Llama 3.3 70B (via Groq) as the core reasoning engine to evaluate profitability, liquidity, and textual sentiment with strict hallucination controls. The system features a robust evaluation framework, benchmarking predictions against historical S&P 500 performance and utilizing a secondary AI judge for logic verification.
>
> **Tech Stack:** `Python`, `LangChain`, `LangGraph`, `Llama 3`, `RAG`, `Vue.js`, `Docker`, `Pytest`, `GitHub Actions`

### 4. [Insurance Cross-Sell: Imbalanced Classification](https://github.com/jose-r-morera/TID-Insurance-Cross-Sell/tree/advanced-classification)
> A machine learning pipeline designed to predict customer cross-selling opportunities from a highly imbalanced insurance dataset. I implemented advanced mitigation techniques, including targeted undersampling, custom class weights, and Focal Loss integration. The project also features categoric model encodings and probability recalibration to align model outputs with real-world distributions.
>
> **Tech Stack:** `Python`, `PyTorch`, `Scikit-learn`, `Pandas`
