### 2. FinanceFlow: Agentic AI Financial Outlook Analyzer (_Currently under Development_)
> A full-stack AI system designed to predict a company's 1-year financial outlook (Positive, Neutral, or Negative) by analyzing SEC filings (10-K, 10-Q, 8-K) through a Retrieval-Augmented Generation (RAG) pipeline. 
> 
> The architecture leverages the Model Context Protocol (MCP) and LangGraph to orchestrate complex reasoning workflows. It utilizes a router-worker model with Llama 3.3 70B (via Groq) as the core reasoning engine to evaluate profitability, liquidity, and textual sentiment without hallucinations. The system includes a robust evaluation framework, benchmarking predictions against historical S&P 500 performance and utilizing a secondary AI judge for logic verification.
>
> **Tech Stack:** `Python`, `Vue.js`, `JavaScript`, `LangChain`, `LangGraph`, `LangSmith`, `Llama 3`, `Groq API`, `RAG`, `Docker`, `Pytest`, `Playwright`, `GitHub Actions`
